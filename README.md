# 🛒 MyOnlineShop - ASP.NET Core MVC & Web API Project

ეს არის სრულფასოვანი ონლაინ მაღაზიის პროექტი, შექმნილი ASP.NET Core MVC და Web API ინტეგრაციით. პროექტი მოიცავს პროდუქციის დათვალიერების, კალათაში დამატების, შეკვეთის გაფორმებისა და ადმინისტრირების ფუნქციონალს.

## 🚀 ძირითადი ფუნქციები

- **პროდუქტების კატალოგი:** მონაცემების დინამიური ჩატვირთვა Web API-დან.
- **პროდუქტის დეტალები:** სრული ინფორმაციის ჩვენება და კალათაში დამატება.
- **კალათის მართვა:** რაოდენობის შეცვლა, წაშლა და **ავტომატური რეცეპტის გენერაცია** ბრაუზერის კონსოლში.
- **შეკვეთის სისტემა:** ვალიდირებული ფორმა, მონაცემების შენახვა ბაზაში და კალათის გასუფთავება წარმატებული შეკვეთის შემდეგ.
- **ადმინ-პანელი:** პროდუქტების CRUD (Create, Read, Update, Delete) ოპერაციები, დამალული "Secret Service" URL-ით.

## 🛠 ტექნოლოგიური სტეკი

- **Framework:** ASP.NET Core 6.0/8.0 MVC
- **Backend:** C#, Entity Framework Core
- **Database:** MS SQL Server
- **Architecture:** N-Tier Architecture, Repository Pattern
- **Frontend:** Razor Views, Bootstrap 5, JavaScript (Vanilla)
- **API:** RESTful Web API

## 📂 არქიტექტურული სტრუქტურა

პროექტი აგებულია **MVC (Model-View-Controller)** პრინციპით:
- **Models:** მონაცემთა ბაზის Entity-ები და ViewModel-ები.
- **Controllers:** Web API-სთან კომუნიკაცია და ბიზნეს ლოგიკის მართვა.
- **Repositories:** მონაცემთა შრის (Data Layer) იზოლაცია კოდის სისუფთავისთვის.

## 🔧 როგორ გავუშვათ პროექტი

1. მოახდინეთ რეპოზიტორის კლონირება: `git clone [თქვენი-ლინკი]`
2. განაახლეთ `appsettings.json`-ში მონაცემთა ბაზის **Connection String**.
3. გაუშვით ბრძანება Package Manager Console-ში: `Update-Database`.
4. დააჭირეთ **F5** პროექტის გასაშვებად.

## 🔒 უსაფრთხოება
ადმინ-პანელზე წვდომა შეზღუდულია და ხელმისაწვდომია მხოლოდ სპეციალურ მისამართზე: `/SecretService`.
